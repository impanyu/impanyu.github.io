  1. http://ec2-54-68-30-150.us-west-2.compute.amazonaws.com:3000 This project implements a chat room application based on node.js, socket.io and express framework.The user can create his name to connect to the chat room. If the user input empty room, the warning of empty name will prompt and the user needs to re-input a valid name. If there are more than 10 people in the room, the user will get notified to wait.If the user goes into the chat room, he can send his messages to everyone in the room, or click "thumbs up" button to send a thumsup icon. Or the user can click the "List All Users" button to take a look at all the users currently online. Each user will have a different color for their messages. If the user does not do anything in the room for more than 15 secs, he will be sent back to the index page.This project is very interesting to me. It demostrates the power of node technology which deals with concurrency with asynchronized IO in a single thread. This is different from multithread or multiplex method which are adopted in other server side technology.
